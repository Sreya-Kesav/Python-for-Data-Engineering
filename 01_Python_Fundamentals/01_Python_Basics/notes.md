## Variables

Variables are just names used to store values.

Example:

```python
name = "Sreya"
age = 24
```

Here,
- `name` stores a string.
- `age` stores an integer.

I can use the variable name whenever I need the value instead of typing the value again.

---

## Naming Variables

Things to remember:

✅ Use meaningful names.

```python
student_name
salary
age
```

❌ Don't start with a number.

```python
1name
```

❌ Don't use spaces.

```python
student name
```

Instead,

```python
student_name
```

## Arithmetic Operators

```python
+
-
*
/
/
//
%
**
```

The ones I should remember well are:

### Floor Division (//)

Returns only the whole number after division.

Example:

```python
103 // 10
```

Output

```
10
```

---

### Modulus (%)

Returns the remainder after division.

Example:

```python
103 % 10
```

Output

```
3
```

Easy way to remember:

```text
// → Quotient

% → Remainder
```
